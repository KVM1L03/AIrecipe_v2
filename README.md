# AIrecipe_v2
Recipe app with recipe generator using ChatGPT assistant and calories calculator.
## Packages used:
* @react-native-async-storage/async-storage
* @react-navigation
* axios
* openai
* react-native-paper
## API:
* OpenAI GPT-3.5 Turbo API
